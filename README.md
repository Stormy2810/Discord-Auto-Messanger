# Discord Auto Messanger

An automated Discord message sender with persistent settings storage. Send messages at regular intervals to any Discord channel.

⚠️ **Disclaimer**: This tool uses Discord's API with user tokens. Automated messaging may violate Discord's Terms of Service. Use responsibly and only in compliance with Discord's guidelines.

## Features

- ⏰ **Customizable intervals** - Set how often messages are sent
- 💾 **Persistent settings** - Saves your configuration locally
- 🔄 **Auto-load** - Remembers your last settings
- 📝 **Simple setup** - Just provide your token, channel ID, and message
- 🚀 **Lightweight** - Minimal dependencies

## Prerequisites

- Python 3.6 or higher
- pip install requests
- Discord account (for user token)
- Channel ID where you have permission to send messages
